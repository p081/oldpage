All about running Mac OS X 10.4 Tiger on unsupported 2008 Macs!

  

Welcome to Project 081

Welcome to Project 081! The sequel to The81Project. It is an unofficial WIP patcher by Apple fans which can allow Mac OS X version 10.4 to run on Early 2008 Apple Macintosh computers. 

 

Is my Mac compatible?

To know if your Mac is compatible with this unofficial distro, click the Apple logo in the- 
menu bar and click About this Mac. If your model is listed as "Early 2008", you can use this tool, although compatibility is not perfect!

Is this legal?

Installing Mac OS X on hackintoshes (Non Apple computers) is against Apple's TOS. However the legality will differ depending on where you live. Installing Mac OS X (even if unsupported) is legal and isn't breaking Apple's TOS because you're still installing it on Apple branded hardware!

Will this patch void my 13 year warranty?

Well, this one is just a joke, unless you actually have a warranty that somehow lasts for 13 years.
Technically, yes but why would Apple enforce such a dumb rule! Even then, Apple never offered a 13 year long warranty so either way: your warranty is void!

How do I install?

Installing is the hardest part, but our goal was always to make it as simple as possible.

It has currently not been released however the first version is work in progress and will be released once enough testing and work has been done. If you wanna install Tiger your 2008 Mac and you can live with worse instructions, messy installs, and Clover bootloader, use The81Project for now.

Whenever it does release, you can be assured testing has been done plenty and the installation process will be simple and painless.
